# Site Garimpo Util

Site/link hub estatico para validacao de canais e credibilidade de programas de afiliados.

## Objetivos

1. Validar canal na Lomadee por metatag.
2. Fortalecer cadastro futuro na Amazon Associados Brasil.
3. Centralizar disclosure e politica editorial.
4. Servir como destino neutro enquanto links de afiliado ainda nao podem ser publicados.

## Publicacao

Pode ser publicado em GitHub Pages, Netlify, Vercel, Cloudflare Pages ou hospedagem estatica equivalente.

Depois de publicar:

1. Usar a URL publica na Lomadee em `Canais de divulgacao`.
2. Conferir se a metatag `lomadee` aparece no HTML publicado.
3. Validar o canal pela Lomadee.
4. Atualizar os documentos operacionais com a URL publica.

## Regra

Nao adicionar links de compra ate a plataforma correspondente estar aprovada, com canal validado e disclosure revisado.
